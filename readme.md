## spustenie Backendu (dev)
```

cd ./backend
mvn spring-boot:run
```


## spustenie Frontendu (dev)
v prečinku frontend 

```
cd ./frontend
npm i
npm run dev
```

## spustenie DB 
```
docker compose up -d
```
